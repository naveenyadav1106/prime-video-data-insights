# Amazon Prime Video Content Analysis

## Project Description
This project analyzes Amazon Prime Video titles and credits to extract valuable insights about content diversity, trends over time, ratings, popularity, and cast/crew contributions. The analysis helps understand which genres dominate, how the library has grown, and which titles, actors, or directors are most prominent.

## Repository Structure
- `/notebook/Amazon_Prime_Video_Content_Analysis.ipynb` — The main Colab notebook with all analyses, visualizations, and insights.  
- `/data/titles.csv` — Dataset containing details of Amazon Prime Video titles.  
- `/data/credits.csv` — Dataset containing cast and director information.

## Libraries Used
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Plotly  

## Key Insights
- Movies dominate the platform, while TV Shows are fewer.  
- Drama and Comedy are the most common genres.  
- The number of titles released per year has grown steadily, peaking in 2021.  
- Certain actors and directors have a large number of titles, highlighting prolific contributors.  
- Outlier titles show exceptional popularity and IMDb ratings, which can inform content strategy.
